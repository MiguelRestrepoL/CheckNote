# 📌 CheckNote  

Proyecto integrador – Miniproyecto N°1.  
Aplicación web de gestión de tareas con tablero Kanban, autenticación y gestión de usuarios.  

## 🚀 Descripción  
CheckNote permite a los usuarios organizar sus tareas de manera sencilla y visual.  
Incluye autenticación JWT, CRUD de tareas, recuperación de contraseña, gestión de perfil y eliminación de cuenta.  
Fue desarrollado como parte de un curso universitario de **Desarrollo Web Full Stack**.  

## 🛠️ Tecnologías  
- **Backend:** Node.js + Express + MongoDB + Mongoose  
- **Arquitectura:** MVC + DAO  
- **Autenticación:** JWT + bcrypt  
- **Frontend:** Inicialmente en React (v2.0), migrado a **HTML, CSS y JS puro** en v3.0  
- **Estilos:** CSS modularizado con variables  

## 📦 Releases  
- **v1.0.0** → Primera versión estable con autenticación, CRUD de tareas, perfil, logging y manejo de errores.  
- **v2.0.0** → Debugging del flujo de “Olvidar contraseña”, mejoras de diseño, corrección del Kanban, frontend en React.  
- **v3.0.0** → Eliminación de tareas y usuario, documentación JSDoc, migración de React a HTML+JS puro, mejoras de contraste y accesibilidad.
